# Python Django 图书管理系统 python70

## 项目概述

本项目是基于 Python Django 框架开发的一个图书管理系统。该系统支持用户登录、注册、登出以及修改密码等基础功能，同时提供图书、作者和出版社信息的添加、查询和管理功能。

## 技术栈

- **后端语言与框架：** Python, Django
- **数据库：** MySQL

## 功能模块

- 用户模块
 - 登录
 - 注册
 - 登出
 - 修改密码
- 图书管理模块
 - 添加图书
 - 查看图书信息
 - 查找出版社信息
 - 添加作者与出版社
- 数据处理
 - 信息唯一性判断
 - 密码加密

## 系统角色

- 系统管理员：负责系统的维护与用户管理
- 普通用户：进行图书查询、信息添加等操作

## 运行环境

- Python 3.x
- Django 3.x
- MySQL 5.x 或更高版本
